# SQL Scripts

This folder contains all SQL Server scripts used for the Target Brazil E-Commerce Analysis project.

The scripts cover:

- Data Exploration and Validation
- Customer Analysis
- Order Trend Analysis
- Seasonality Analysis
- State-wise Performance Analysis
- Freight Cost Analysis
- Delivery Performance Analysis
- Payment Behavior Analysis

Key SQL concepts demonstrated:

- Joins
- Aggregate Functions
- Date Functions
- CASE Statements
- Common Table Expressions (CTEs)
- Group By and Having Clauses
- Subqueries
- Business-Oriented Data Analysis

The queries were designed to answer business questions and generate actionable insights from the e-commerce dataset.
