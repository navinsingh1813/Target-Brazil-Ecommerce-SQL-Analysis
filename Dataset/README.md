# Dataset Information

Dataset: Brazilian E-Commerce Public Dataset by Olist

Tables Used:
- customers
- orders
- order_items
- payments
- products
- sellers
- geolocation
- reviews

Note:
The geolocation dataset has not been uploaded to this repository due to GitHub web upload size limitations. The complete dataset can be downloaded from the original source.

Dataset Source:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
