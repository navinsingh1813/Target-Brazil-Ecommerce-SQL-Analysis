# Problem Statement

This folder contains the original business problem statement provided for the Target Brazil E-Commerce Analysis project.

The objective of the project is to analyze Target Brazil's e-commerce dataset and extract actionable business insights related to:

- Customer Behavior
- Order Trends
- Monthly Seasonality
- State-wise Performance
- Freight Cost Analysis
- Delivery Performance
- Payment Patterns

The analysis was performed using SQL Server by querying multiple relational tables and answering business-focused analytical questions.
